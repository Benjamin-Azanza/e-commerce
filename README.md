#Cap & Knit

## Características Principales

* *Visualización de Productos*: Muestra una cuadrícula de productos desde un objeto estático de JavaScript.
* *Detalle de Producto*: Cada producto tiene una página de detalle individual, accesible a través de un clic. 
* *Carrito de Compras Persistente*:
    * Añadir productos al carrito desde la página de detalle.
    * Ver el carrito en una página dedicada.
    * Modificar cantidades (sumar, restar).
    * Eliminar productos del carrito.
    * El carrito se guarda en localStorage para que no se borre en cada recarga de página, o cuando el usuario cambie entre paginas.
* *Contador de Carrito*: El icono del carrito en el header se actualiza en tiempo real.
  
* *Diseño Responsivo*: Creado con Bootstrap importando clases para adaptarse a dispositivos móviles y de escritorio y utilizar una plantilla para hacer la pagina web.

## Tecnologías Utilizadas

* *HTML5*: Estructura semántica del sitio web.
* *CSS*: Estilos personalizados menores (principalmente para los mensajes de formulario).
* *Bootstrap 5*: Framework principal para el layout, componentes (tarjetas, botones, formularios) y diseño responsivo.
* *JavaScript *: Controla toda la lògica de la aplicaciòn.
* *jQuery*: Librería principal para la manipulación del DOM y manejo de eventos.
* *Git & GitHub*: Control de versiones y hospedaje del código.
